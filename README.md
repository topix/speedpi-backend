SpeedPi-Backend
==============

**Gateway**
We use [MicroServiceBus.com](https://microservicebus.com) to deploy and manage nodes in our network. 

**Data storage**
We use MySQL to store our data for later analys.

**Data analys**
We plan to use TensorFlow to analyze the health of:

 - Each area (over time)
 - Each node (over time)
 - ISP (over time)
